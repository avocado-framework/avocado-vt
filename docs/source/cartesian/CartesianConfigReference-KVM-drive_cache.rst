
drive\_cache
============

Description
-----------

Sets the caching mode a given drive. Currently the valid values are:
writethrough, writeback, none and unsafe.

Example:

::

    drive_cache = writeback

This option can also be set specifically to a drive:

::

    drive_cache_cd1 = none
